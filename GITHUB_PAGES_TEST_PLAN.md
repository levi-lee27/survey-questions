# GitHub Pages 部署测试计划

## 前提条件
- [x] 代码已推送到 GitHub 仓库: levi-lee27/survey-questions
- [ ] GitHub Pages 已启用（需要用户操作）

## 第一步：启用 GitHub Pages

访问：https://github.com/levi-lee27/survey-questions/settings/pages

1. 在 "Build and deployment" 部分
2. Source 选择: **Deploy from a branch**
3. Branch: **main** 分支，目录 **/ (root)**
4. 点击 **Save**
5. 等待 1-2 分钟（会显示绿色构建状态）

## 第二步：验证部署成功

构建完成后，访问：
```
https://levi-lee27.github.io/survey-questions/
```

应该看到项目文件列表或直接进入 index.html。

## 如果发现问题

### 问题：Pages 显示 404
**原因**：还在构建中  
**解决**：等待 2-3 分钟，刷新页面

### 问题：样式加载失败
**原因**：GitHub Pages 缓存  
**解决**：强制刷新 Ctrl+F5 或 Shift+F5

### 问题：无法提交问卷
**检查**：
1. 浏览器控制台是否有错误
2. localStorage 是否可用（Settings → Privacy → Cookies）
3. surveyId 参数是否正确传递
