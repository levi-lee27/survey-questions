【步骤 1：创建 Firebase 项目】
   URL: https://console.firebase.google.com/
   □ 登录 Google 账号
   □ 创建项目（名称：survey-questions）
   □ 等待项目创建完成（1-2分钟）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【步骤 2：启用 Realtime Database】
   □ 点击 "Realtime Database"
   □ 点击 "Create Database"
   □ 选择位置（如 asia-southeast1）
   □ 选择 "Start in test mode"
   □ 点击 "Enable"
   □ 复制数据库 URL（备用）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【步骤 3：获取 Web 应用配置】
   □ 点击项目名称 → ⚙️ Project settings
   □ 滚动到 "Your apps"
   □ 点击 Web 图标 "</>"
   □ App nickname: survey-web
   □ 不勾选 "Also set up Firebase Hosting"
   □ 点击 "Register app"
   □ 复制整个 firebaseConfig 对象

   示例格式：
   const firebaseConfig = {
     apiKey: "AIzaSy...",
     authDomain: "...",
     databaseURL: "https://...",
     projectId: "...",
     storageBucket: "...",
     messagingSenderId: "...",
     appId: "..."
   };

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【步骤 4：填入配置文件】
   文件：C:\Users\levi_\openclaw\survey-web\firebase-config.js

   □ 用记事本打开 firebase-config.js
   □ 删除所有示例注释
   □ 粘贴你复制的配置
   □ 保存文件

   正确示例：
   const firebaseConfig = {
     apiKey: "AIzaSyXXXXXXXXXXXXXXXXX",
     authDomain: "survey-questions.firebaseapp.com",
     databaseURL: "https://survey-questions-default-rtdb.asia-southeast1.firebasedatabase.app",
     projectId: "survey-questions",
     storageBucket: "survey-questions.appspot.com",
     messagingSenderId: "123456789012345",
     appId: "1:123456789012345:web:abcdef123456"
   };

【步骤 5：验证配置（可选但推荐）】

   方法 A - 使用验证工具：
   □ 启动本地服务器：
     cd C:\Users\levi_\openclaw\survey-web
     python -m http.server 8080
   □ 浏览器访问：http://localhost:8080/check-firebase-config.html
   □ 粘贴配置代码，点击验证

   方法 B - 直接测试：
   □ 访问：http://localhost:8080/generate.html
   □ 按 F12 打开控制台
   □ 查看是否显示 "[Firebase] 初始化成功"

【步骤 6：设置安全规则】

   Firebase Console → Realtime Database → Rules

   测试阶段使用（允许所有人读写）：
   {
     "rules": {
       "surveys": {
         ".read": true,
         ".write": true
       }
     }
   }

   □ 粘贴规则代码
   □ 点击 "Publish"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【步骤 7：测试跨设备同步】

   电脑端：
   □ 访问 https://levi-lee27.github.io/survey-questions/generate.html
   □ 创建问卷（例如标题："测试问卷"）
   □ 填写并提交 2-3 次
   □ 记录 surveyId（从 URL 或生成器页面）

   手机端：
   □ 打开微信扫一扫，扫描问卷二维码
   □ 或直接访问问卷链接
   □ 填写并提交 1-2 次

   电脑端再次验证：
   □ 访问 https://levi-lee27.github.io/survey-questions/admin.html?surveyId=你的surveyId
   □ 看到手机提交的数据 ✅
   □ 右上角显示 "● 云端同步"（绿色）✅
   □ 提交新问卷时右下角出现通知 ✅
